# UX Principles

1. **Claridad sobre cleverness.** Si dos opciones existen, elegí la más obvia.
2. **Estado siempre visible.** El usuario nunca debería preguntarse "¿pasó algo?".
3. **Recuperación primero, prevención después.** Diseñá para el error: cómo se sale, no solo cómo se evita.
4. **Mobile real, no mobile mock.** Probá en un teléfono físico antes de declarar responsive.
5. **Performance es UX.** 1s de carga vs 3s cambia el comportamiento, no solo la percepción.
6. **Consistencia > novedad.** Reusar patrones conocidos baja la carga cognitiva.
7. **Accesibilidad por default.** Diseñar accesible es más barato que retrofittear.
