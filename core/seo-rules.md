# Reglas de SEO y Metadatos — UX Harness v3.1

> Todo proyecto debe cumplir con estos estándares para superar el Test 08.

---

## 1. Etiquetas de Título (Title Tags)
- **Longitud**: Entre 50 y 60 caracteres.
- **Estructura**: `[Palabra Clave] | [Nombre de Marca]`.
- **Único**: Cada página debe tener un título único.

## 2. Meta Descripciones
- **Longitud**: Entre 150 y 160 caracteres.
- **Contenido**: Debe incluir un Call to Action (CTA) y resumir el valor de la página.
- **Voz**: Coincidir con el `tone_and_voice` del cliente.

## 3. OpenGraph (Social Media)
Obligatorio para mejorar el CTR en compartidos:
- `og:title`: Título optimizado para redes.
- `og:description`: Descripción breve.
- `og:image`: Imagen de 1200x630px representativa.
- `og:type`: `website` o `article`.

## 4. Jerarquía de Encabezados (Hn)
- Un solo `<h1>` por página.
- Estructura lógica descendente (H1 -> H2 -> H3).
- No saltar niveles (ej: de H1 a H3).

## 5. Accesibilidad SEO
- Todas las imágenes deben tener atributo `alt` descriptivo.
- Los enlaces deben tener texto descriptivo ("Ver detalles de propiedad", no "Click aquí").
